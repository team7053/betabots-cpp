Welcome!
This is the C++ conversion of Team 7053 Robo Huskies Betabots code originally
written in Labview written by Nathan Salmard.

This project requires the WPILib 2019 and the Kauai Labs NavX Library for
the NavX Gyroscope.

Known issues:
PID system will no longer work with the WPILib 2020 update
